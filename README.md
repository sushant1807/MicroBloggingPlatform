# MicroBloggingPlatform
MVVM with Kotlin

This repository contains a detailed client-server application that implements in Model-View-ViewModel

Libraries and Tools: 

The sample app is using libraries and frameworks:

1. View Model 
2. Live data 
3. Data Binding 
4. Room DB 
5. RXJava
6. RXAndroid 
7. Dagger 2
8. Retrofit 
9. OKHTTP
10. Material Design
